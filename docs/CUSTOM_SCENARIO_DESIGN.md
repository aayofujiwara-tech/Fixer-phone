# FIXER PHONE — カスタムシナリオ機能 設計書

## 概要

ユーザーが自分でシナリオを作成し、FIXER PHONEに導入して遊べるようにする機能。
シナリオの生成はChatGPTやClaude等の外部AIを利用し、アプリ側はテキストベースでのインポート/エクスポートに特化する。サーバー不要、PWAのまま完結。

## コンセプト

- **CREATE MISSION** — ユーザーが自分だけの極秘ミッションを作成
- 全てテキストのコピペで完結（ファイルアップロード不要）
- AIにプロンプトを投げてシナリオを生成 → JSONをアプリに貼り付け
- 保存済みシナリオは通常シナリオと同じように再生可能

## ユーザーフロー

### シナリオ作成フロー

```
セットアップ画面
  → 「CREATE MISSION」ボタン
  → 作成画面が開く
    ├── ① フォーマット説明（JSONの構造を表示）
    ├── ② テーマ提案（選択 or 自由入力）
    ├── ③ AI用プロンプト（コピー用、ワンタップでコピー）
    ├── ④ ユーザーが外部AIでシナリオ生成
    ├── ⑤ 生成されたJSONをテキストエリアに貼り付け
    ├── ⑥ バリデーション（フォーマットチェック）
    ├── ⑦ プレビュー（シナリオ名・最初のセリフ等を確認）
    └── ⑧ 保存
```

### シナリオ再生フロー

```
セットアップ画面
  → 「CUSTOM」枠（12カ国グリッドの下 or 別タブ）
  → 保存済みシナリオ一覧
  → シナリオ選択
  → AUTO / PRACTICE 選択
  → 通話開始（通常シナリオと同じ再生フロー）
```

### シナリオ管理フロー

```
CUSTOM枠内
  → 保存済みシナリオ一覧
    ├── 再生
    ├── 書き出し（JSONテキストをコピー → 別端末に移行可能）
    └── 削除
```

## シナリオJSONフォーマット

既存シナリオと同一構造。ユーザーに提示するテンプレート:

```json
{
  "id": "custom_001",
  "title": {
    "ja": "シナリオタイトル（日本語）",
    "en": "Scenario Title (English)"
  },
  "country": "CUSTOM",
  "mood": "serious",
  "vip": {
    "role": {
      "ja": "大統領",
      "en": "President"
    },
    "icon": "🏛",
    "title": {
      "ja": "閣下",
      "en": "Mr. President"
    }
  },
  "lines": [
    {
      "speaker": "vip",
      "ja": "VIPの最初のセリフ（日本語）",
      "en": "VIP's first line (English)"
    },
    {
      "speaker": "fixer",
      "ja": "フィクサーの返答（日本語）",
      "en": "Fixer's response (English)"
    },
    {
      "speaker": "vip",
      "ja": "VIPの2番目のセリフ",
      "en": "VIP's second line"
    },
    {
      "speaker": "fixer",
      "ja": "フィクサーの返答",
      "en": "Fixer's response"
    },
    {
      "speaker": "vip",
      "ja": "VIPの3番目のセリフ",
      "en": "VIP's third line"
    },
    {
      "speaker": "fixer",
      "ja": "フィクサーの返答",
      "en": "Fixer's response"
    },
    {
      "speaker": "vip",

      "en": "Fixer's response"
    },
    {

      "ja": "VIPの5番目のセリフ",
      "en": "VIP's fifth line"

    {
      "speaker": "fixer",
      "ja": "フィクサーの最後の返答",
      "en": "Fixer's final response"
    }

}
```

## バリデーションルール

- `lines` 配列が10要素（5往復）であること
- 奇数番目（0,2,4,6,8）が `speaker: "vip"`、偶数番目（1,3,5,7,9）が `speaker: "fixer"`
- 各セリフに `ja` と `en` のテキストが存在すること
- `title` に `ja` と `en` が存在すること
- `vip.role` / `vip.title` に `ja` と `en` が存在すること
- `mood` が `"serious"` または `"comedy"` であること

## テーマ提案例

ユーザーに表示するテーマのサンプル:

### SERIOUS（ガチ交渉）

| テーマ | 概要 |

| 🛸 宇宙人との外交 | 未知の文明とのファーストコンタクト交渉 |

| 🌋 超巨大噴火 | 破局噴火による国際避難計画の交渉 |

### COMEDY（コメディ）

| テーマ | 概要 |
|--------|------|
| 🍣 国際寿司サミット | 各国が「本物の寿司」の定義で対立 |

※ユーザーは上記から選んでもよいし、完全オリジナルのテーマを自由に設定してもよい。

## AI用プロンプト（コピー用）

ユーザーが外部AI（ChatGPT / Claude等）に貼り付けて使うプロンプト:

```
あなたはバイリンガル・ロールプレイゲーム「FIXER PHONE」のシナリオライターです。
以下の設定でシナリオを1本作成してください。

【テーマ】: （ここにテーマを入力）
【モード】: serious / comedy（どちらか選択）
【VIPの役職】: （例: 大統領、首相、皇帝、宇宙艦隊司令官 等）
【VIPの敬称】: （例: 閣下/Mr. President、殿下/Your Highness 等）

【ルール】
- 5往復10セリフ（VIP→フィクサー→VIP→...→フィクサー）
- VIPが先に話し、フィクサーが最後に話す
- VIPの口調: 命令・断定調（「〜だ」「〜してくれ」）
- フィクサーの口調: 丁寧語（「〜です」「〜ます」）冷静・プロフェッショナル
- 各セリフに日本語（ja）と英語（en）を併記
- 日本語は自然な会話体、英語も自然な会話体
- 1セリフは2〜4文程度

以下のJSONフォーマットで出力してください:

{
  "id": "custom_001",
  "title": {
    "ja": "シナリオタイトル",
    "en": "Scenario Title"
  },
  "country": "CUSTOM",
  "mood": "serious または comedy",
  "vip": {
    "role": { "ja": "役職名", "en": "Role" },
    "icon": "🏛",
    "title": { "ja": "敬称", "en": "Honorific" }
  },
  "lines": [
    { "speaker": "vip", "ja": "...", "en": "..." },
    { "speaker": "fixer", "ja": "...", "en": "..." },
    { "speaker": "vip", "ja": "...", "en": "..." },
    { "speaker": "fixer", "ja": "...", "en": "..." },
    { "speaker": "vip", "ja": "...", "en": "..." },
    { "speaker": "fixer", "ja": "...", "en": "..." },
    { "speaker": "vip", "ja": "...", "en": "..." },
    { "speaker": "fixer", "ja": "...", "en": "..." },
    { "speaker": "vip", "ja": "...", "en": "..." },
    { "speaker": "fixer", "ja": "...", "en": "..." }
  ]
}

JSONのみを出力してください。説明は不要です。
```

## 保存方式

- **localStorage** に保存
- キー: `fixer-phone-custom-scenarios`
- 値: カスタムシナリオの配列（JSON文字列）
- 上限: 10シナリオ程度（localStorageの容量制約を考慮）

## UI配置

### セットアップ画面への統合

```
[SPEAKER/STEALTH] ← 音量ボタン（既存）

[🎲 RANDOM MISSION] ← 全ランダム（既存）

[🇯🇵] [🇺🇸] [🇨🇳] [🇷🇺] [🇬🇧] [🇫🇷]   ← 12カ国グリッド（既存）
[🇩🇪] [🇰🇷] [🇮🇱] [🇮🇳] [🇸🇦] [🇻🇦]

[＋ CREATE MISSION]   ← 新規作成ボタン（新規）
[📁 CUSTOM (3)]       ← 保存済みカスタムシナリオ一覧（新規、件数表示）
```

### 世界観との統合

- 「CREATE MISSION」=「自分で極秘ミッションを作成」
- 「CUSTOM MISSION」=「独自の極秘作戦」
- UIは既存のスパイ/作戦室風デザインを踏襲
- カスタムシナリオ再生時のVIPアバターは汎用のシルエット（国旗なし）

## 実装優先度

| 優先度 | 機能 |
|--------|------|
| P0 | JSONテキスト貼り付け → バリデーション → 保存 |
| P0 | 保存済みシナリオの一覧 → 選択 → 再生 |
| P1 | AI用プロンプトのコピー機能 |
| P1 | テーマ提案の表示 |
| P1 | シナリオの書き出し（JSONコピー）/ 削除 |
| P2 | テーマ提案からプロンプトへの自動挿入 |
| P2 | VIPの音声パラメータのカスタマイズ |

## 技術的な注意点

- 既存のシナリオデータ構造との互換性を維持すること
- CallScreen は `country="CUSTOM"` の場合、国旗の代わりに汎用アイコンを表示
- カスタムシナリオのVIP音声パラメータはデフォルト値を使用（将来カスタマイズ可能に）
- localStorage の容量オーバーをハンドリングすること（try-catch）
